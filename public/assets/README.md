# Logo

Coloca aquí el logo de EcoValue con el nombre **`logo.png`**.

- Formato recomendado: **PNG con fondo transparente**
- Ancho recomendado: **~480px** (se ajusta automáticamente)
- Ruta que usa la página: `/assets/logo.png`

Mientras no exista `logo.png`, la página muestra un recuadro punteado
que reserva el espacio ("Coloca aquí el logo de EcoValue"), así el diseño
no se desacomoda.

> Si tu logo tiene otro nombre o extensión (por ejemplo `logo.svg`),
> actualiza el atributo `src` en `public/index.html`.
